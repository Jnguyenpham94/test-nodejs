Instructions

Register Gesture

    Steps to register a gesture to program
  1. Turn on camera.
  2. Raise hand gesture to camera.
  3. Press Register Gesture button.
  4. Naming window will show up.
  5. Give memorable name and press save.
  6. Keep gesture up for the 5 second count.
  7. A new gesture in Profile Menu with the created name will now be available to be used in a profile.

Profile Menu

From here a profile can be chosen, created or editted.

Actions Menu

From here actions can be created to later be mapped to a profile and gesture in the Profiles Menu.

General Program use

After choosing a profile from the profile menu with the desired gesture mappings. Turn on camera and hold up the gesture mapped to that profile to do the desired function.

If the gesture being read is not being recognized as quickly or is being too generous then go into the settings and modify the confidence value. Closer the number to 1, the more strict the gesture will be compared to the gesture in the profile.
