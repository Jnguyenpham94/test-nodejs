Instructions

Profile Menu

From here a profile can be chosen, created or editted.

Register Gesture

    Steps to register a gesture to program
  1. Turn on camera.
  2. Raise hand gesture to camera.
  3. Press Register Gesture button.
  4. Naming window will show up.
  5. Give memorable name and press save.
  6. Keep gesture up for the 5 second count.
  7. A new gesture in Profile Menu with the created name will now be available to be used in a profile.

Actions Menu

From here actions can be created to later be mapped to a profile and gesture in the Profiles Menu.

From here you can choose a file to map to a gesture.

From here a button shortcut can be mapped and saved for a gesture.

General Use

After choosing a profile from the profile menu with the desired gesture mappings OR using the create profile button to create your own named profiles and mappings. Turn on camera and hold up the gesture mapped to that profile to do the desired function.

If the gesture is not being recognized or is being too generous in recognizing the gesture then go into the settings and modify the Hand Match Confidence.

  1. Closer to 1, the more strict the gesture will be compared to the gesture in the profile.
  2. Closer to 0, more strict it will be in recognizing a particular gesture.
  3. NOTE: we found a number from .6 to .75 to be the sweet spot.
