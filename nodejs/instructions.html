Register Gesture

    Steps to register a gesture to program
  1. Turn on camera.
  2. Raise hand gesture to camera.
  3. Press Register Gesture button.
  4. Naming window will show up.
  5. Give memorable name and press save.
  6. Keep gesture up for the 5 second count.
  7. A new gesture in Profile Menu with the created name will now be available in a profile.

General Program use

After choosing a profile from the profile menu with the desired gesture mappings. Turn on camera and hold up the gesture mapped to that profile to do the desired function.
